h1. ModulManager

Prüfungsordnung leicht gemacht -- Der ModulManager ist eine hochflexible Webanwendung zur Abbildung von Prüfungsordnungen, mit der Studierende ihr Studium einfacher planen können. Durch einen regelbasierten Ansatz lässt sich die Open-Source-Software ModulManager für jeden Studengang anpassen.
